## 改造 `MultiSpecSheet.tsx` 详细规格 + 批量设置

### 1. 详细规格区（重做布局，对齐截图1）

去掉当前 `grid-cols-[1.5fr_1fr_1fr_1fr_28px]` 紧凑横排，改成「**组合名独占一行 + 下方一行四列输入**」：

```
颜色/尺码                              ○ 显示商品编码
价格*        成本价        库存        图片
黑色白标/M（80-100斤）
[请输入]    [请输入]      [不限]      [ + ]
黑色白标/L（100-120斤）
[请输入]    [请输入]      [不限]      [ + ]
...
```

- 顶部一行：左侧用所有规格组名拼成 `颜色/尺码`，右侧保留「显示商品编码」单选圈样式（不是原生 checkbox）。
- 列头：`价格*` / `成本价` / `库存` / `图片`，灰色小字 11px。
- 每个 SKU：组合名一行（13px 加粗），输入框一行 4 列，`grid-cols-[1fr_1fr_1fr_56px]`，图片格 56×40 虚线 `+`。
- 库存 placeholder 改为 `不限`，价格红 `*`。
- 移除当前每行内 toast 占位图标外的边框/灰底。

「批量设置」按钮从详细规格头部右上角触发（绿色文字，对齐截图1）—— 不再在卡片内部展示批量输入条。

### 2. 批量设置弹层（新建，对齐截图2）

在 `MultiSpecSheet` 内新增一个底部弹层 `BatchSettingSheet`（局部 absolute，不用 Radix，遮罩半透明灰）：

结构：
- 顶部：`批量设置` 标题 + 右上 `×`。
- **选择规格 (可多选)**：按 `groups` 渲染，每组一行：左侧标签（如「颜色」），右侧 chips：`全部` + 每个 value。
  - chip 选中态：浅绿底 `#e8f7ee` + 绿字 `#07c160` + 绿边；未选灰底 `#f4f5f7`。
  - 点 `全部` = 选中该组所有 value（toggle）。
  - 每组必须至少 1 个被选才能"确定"。
- **对已选规格批量设置**：
  - 价格（右侧"元"后缀）
  - 成本价（"元"后缀）
  - 库存（右侧"不限"单选圈，勾上则忽略输入）
  - 图片（占位 `+` 方块）
- 底部绿色 `确定` 按钮（全宽贴底）。

确定逻辑：对每个 `variant`，若其每个组的 valueId 都 ∈ 该组被选集合，则套用价格/成本价/库存（仅写非空字段；库存若勾「不限」则写空串）。完成后关闭弹层 + toast 成功。
